ExSize is a web app where parents assign chores to children and reward them with virtual currency (ExBucks). Children earn ExBucks, buy rewards, and progress through a gamification system (levels, streaks, badges).

Key constraints:
- Zero infrastructure budget (free tiers only)
- Web-only MVP (iOS post-MVP)
- Bilingual from day one (PL + EN)
- No tech stack chosen yet — needs to be decided before first implementation

**Why:** Greenfield project, PRD lives in issue #1, plan in `plans/exsize-mvp.md`
**How to apply:** Always check issue blockers before starting a phase. Tech stack decision is pending.
